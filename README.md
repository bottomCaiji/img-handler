### 使用方式
* 直接将多个图片或文件夹拖拽到.exe文件上就可以了
* 如果图片尺寸较大/较多, 可能拼接会有些慢, 请耐心等候
* 【ctrl + C】可终止程序

### feature
* 支持【".png", ".jpg", ".jpeg", ".bmp", ".ico", ".tga", ".tiff"】后缀的文件
* 支持 瀑布流 和 傻瓜连排 布局
* 支持 水平 和 竖直 排列
