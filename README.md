# img-handler
# 图片拼接工具

本程序仅支持.png、.jpg格式图片的拼接，所有图片必须为相同格式，以下用.png举例，.jpg同理   

将所有待拼接图片放到一个文件夹，依次命名为 '行-列.png'   

例如你想将6张图片拼接成3列，那么应该依次命名为 '1-1.png', '1-2.png', '1-3.png', '2-1.png', '2-2.png', '3-3.png'   

输出的图片保存在当前目录，文件名为'result.png'
